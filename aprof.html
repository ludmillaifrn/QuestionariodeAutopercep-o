Com a chegada do Setembro Amarelo, a psicóloga do IFCM pediu ajuda
aos professores para que os mesmos desenvolvessem atividades relacionadas
a temática do mês, que é a saúde mental, o professor Gilbran, que ministra
a matéria de programação com acesso a banco de dados (PABD) dividiu a turma
em grupos para que realizassem algum projeto que se relacionasse a temática,
assim foi criado o" Questionário de Autopercepção", onde os alunos respondem
algumas simples perguntas e recebem uma pontuação e uma mensagem que indica
se eles deveriam procurar ajuda profissional.
